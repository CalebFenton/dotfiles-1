# RichGuk's dotfiles

My dotfiles/setup scripts that are used on variables machines I use.


## Setup

1. Clone repo

```
git clone https://github.com/RichGuk/dotfiles.git .dotfiles
```

2. Run symlinks

```
cd .dotfiles
./bin/setup.sh
```

3. Mac only setup (installs applications, and configures settings)

```
./bin/mac_setup.sh
```

4. Done.
